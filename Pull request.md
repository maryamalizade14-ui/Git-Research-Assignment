۴. Pull Request (پر - درخواست ادغام)

· چیه؟ درخواست رسمی به صاحب پروژه برای اینکه تغییرات تو رو در پروژه اصلی ادغام (merge) کنه.
· کاربرد: بعد از Fork و ایجاد تغییر، می‌خواهی کارت را به پروژه اصلی اضافه کنی.
· مراحل:
  ۱. پروژه رو Fork کن
  ۲. تغییرات رو ایجاد کن
  ۳. در GitHub روی Pull Request کلیک کن
  ۴. صاحب پروژه بررسی و تأیید می‌کنه
