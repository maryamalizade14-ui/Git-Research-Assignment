۵. تفاوت Pull، Pull Request و Push

عمل انجام‌دهنده جهت توضیح
Pull تو از GitHub به کامپیوترت گرفتن تغییرات دیگران
Push تو از کامپیوترت به GitHub فرستادن تغییرات خودت
Pull Request تو → صاحب پروژه درخواست ادغام درخواست برای اضافه شدن کارت به پروژه اصلی

مثال:

· دوستت تغییراتی داد → تو pull می‌کنی
· تو تغییراتی دادی → push می‌کنی
· می‌خواهی تغییرات تو به پروژه اصلی اضافه بشه → pull request می‌سازی
